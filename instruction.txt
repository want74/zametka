Задача: "Спроектировать свой сайт".

1 урок:
	1. Выбор темы проекта (10 минут)
	"Материалы" -> "64" -> Открыть файл themes.txt
	Утверждение темы проекта.

	2. Определение функционала сайта
		a. Создать файл site.txt на сервере. Ответить по простому на 2 вопроса: (10 минут)
			1. "Для чего нужен твой сайт?"
			2. "Кто им будет пользоваться?"

			Например: сайт афиши.
			1. Чтобы люди, могли узнать что идет в кинотеатрах, театрах, концертных залах. Где, когда идет и сколько стоит. 
			2. Люди, которые любят смотреть фильмы, спектакли и концерты.
			Утверждение тем.

		b. Бэкенд сайта (20-30 минут)
			"Материалы" -> "64" -> Открыть файл example_backend. Пример схемы бэкэнда.
			Изучить сайты с похожей тематикой (как у твоего проекта). На листочках нарисовать и сфоткать схему проекта.
			1. Какие функции есть на сайте?
			2. Какие запросы будут?
			Утверждение бэкенда.

		c. Архитектура БД
			"Материалы" -> "64" -> Открыть файл example_architecture. Пример архитектуры БД (15-20 минут)
			Составить архитектуру БД и таблиц в ней. На листочках нарисовать и сфоткать 
			Ответьте для себя на вопросы: 
			1. Сколько нужно таблиц?
			2. Сколько столбцов в таблицах?
			3. Какой тип данных у столбцов?
			4. Есть ли объединение нескольких таблиц?
			Утверждение архитектуры.
